# **App Name**: DataInsights AI

## Core Features:

- Dataset Detection: Automatically detect the uploaded dataset format (CSV, Excel) and parse it accordingly.
- Automated Data Cleaning: Automatically identify and handle missing values, outliers, and inconsistencies in the dataset.
- Automated EDA: Perform basic EDA (Exploratory Data Analysis) like summary statistics, distributions, and correlations on the dataset.
- Analysis Suggestion Tool: Suggest potential analysis types (regression, classification, clustering) based on the dataset's characteristics and provide tool to decide whether or not each type would be useful.
- Automated Preprocessing: Automatically perform necessary preprocessing steps like scaling, encoding, and feature selection based on the selected analysis type.
- Model Analysis & Visualization: Perform the selected analysis type and visualize the results using appropriate charts and graphs.
- Results Presentation: Display analysis results in an easy-to-understand and interactive format.

## Style Guidelines:

- Primary color: Navy blue (#1A237E) to convey a sense of professionalism and data focus.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop for the data and visualizations.
- Accent color: Teal (#008080) to highlight key insights and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font, for a clean, modern and objective look.
- Use clear, data-focused icons to represent different analysis types and data transformations.
- Employ a grid-based layout for organized data presentation and easy navigation.
- Use subtle transitions and animations to guide the user through the analysis workflow.
